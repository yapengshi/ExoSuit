# FeetechMotor read joint angles
Some Test code on FeetechMotor using the Feetech Servo SDK to read joint angles.

## FeetechMotor.py
please use `python3.10` or above to run this code.

dependecies:
- `pip3 install feetech-servo-sdk`

```bash
sudo chmod 666 /dev/ttyACM0 # permission to access the FeetechMotor
python3 FeetechMotor.py
```
